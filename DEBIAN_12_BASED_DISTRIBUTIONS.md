# Linux Distributions Based on Debian 12 (Bookworm)

This document lists Linux operating systems and distributions that are built on or compatible with Debian 12 (Bookworm), the stable release of Debian GNU/Linux.

## üéØ Popular Debian 12-Based Distributions

### Ubuntu 24.04 LTS (Noble Numbat) and Later
- **Description**: Based on Debian Sid but inherits from Debian 12 packages
- **Base**: Debian Sid (Unstable) with Debian 12 core packages
- **Desktop**: GNOME (default), KDE, Xfce, MATE available
- **Focus**: User-friendly desktop and server Linux
- **Website**: https://ubuntu.com

### Linux Mint (Latest Versions)
- **Description**: User-friendly distribution based on Ubuntu/Debian
- **Base**: Ubuntu LTS (which uses Debian packages) or Debian directly (LMDE)
- **Desktop**: Cinnamon (flagship), MATE, Xfce
- **Focus**: Desktop usability and elegance
- **Website**: https://linuxmint.com

### Linux Mint Debian Edition (LMDE)
- **Description**: Linux Mint directly based on Debian
- **Base**: Debian 12 (Bookworm)
- **Desktop**: Cinnamon
- **Focus**: Debian-direct version without Ubuntu layer
- **Website**: https://www.linuxmint.com/download_lmde.php

### MX Linux
- **Description**: Midweight desktop distribution
- **Base**: Debian Stable (Bookworm)
- **Desktop**: Xfce (default), KDE Plasma, Fluxbox
- **Focus**: Efficiency and user-friendliness
- **Website**: https://mxlinux.org

### Kali Linux
- **Description**: Security and penetration testing distribution
- **Base**: Debian Testing (with Bookworm foundations)
- **Desktop**: Xfce, GNOME, KDE, MATE, others
- **Focus**: Security auditing, penetration testing, forensics
- **Website**: https://www.kali.org

### antiX
- **Description**: Fast, lightweight systemd-free distribution
- **Base**: Debian Stable (Bookworm)
- **Desktop**: IceWM, Fluxbox, JWM, Herbstluftwm
- **Focus**: Old hardware, speed, no systemd
- **Website**: https://antixlinux.com

### Devuan
- **Description**: Debian fork without systemd
- **Base**: Debian (Daedalus is based on Bookworm)
- **Desktop**: Multiple desktop environments
- **Focus**: Init freedom (sysvinit, OpenRC, runit)
- **Website**: https://www.devuan.org

### Raspberry Pi OS (Bookworm)
- **Description**: Official OS for Raspberry Pi
- **Base**: Debian 12 (Bookworm)
- **Desktop**: Modified LXDE/Pixel
- **Focus**: Raspberry Pi hardware optimization
- **Architecture**: arm64, armhf
- **Website**: https://www.raspberrypi.com/software/

### PureOS
- **Description**: Fully free (FSDG-compliant) distribution
- **Base**: Debian with only free software
- **Desktop**: GNOME
- **Focus**: Freedom, privacy, security
- **Website**: https://www.pureos.net

### SparkyLinux
- **Description**: Lightweight, fast distribution
- **Base**: Debian Testing/Stable (Bookworm)
- **Desktop**: Xfce, LXQt, KDE, MATE, others
- **Focus**: Speed and simplicity
- **Website**: https://sparkylinux.org

## üîÑ Why Debian 12?

Distributions choose Debian 12 (Bookworm) as their base because:

1. **Stability**: Debian Stable releases are thoroughly tested
2. **Security**: Regular security updates and long-term support
3. **Package Management**: APT and dpkg are mature and reliable
4. **Documentation**: Extensive documentation and community support
5. **Freedom**: Debian Free Software Guidelines (DFSG)
6. **Multi-architecture**: Support for many CPU architectures
7. **Community**: Large, active community of developers and users
